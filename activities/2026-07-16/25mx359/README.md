# Mission Tata 1 – Git & Collaboration

## Objective

The objective of this mission was to understand and practice the complete Git workflow.

## Tasks Completed

* Forked the main repository.
* Cloned the repository to the local machine.
* Created a new folder using my roll number.
* Added the required profile document.
* Committed the changes.
* Pushed the changes to my GitHub fork.
* Opened a Pull Request to the main repository.

## Git Commands Used

```bash
git clone <repository-url>

git checkout -b mission-tata-1

git add .

git commit -m "Added profile and mission documents"

git push origin mission-tata-1
```

## Learning Outcome

This activity helped me understand the complete Git collaboration workflow, including version control, branching, committing, pushing, and creating pull requests.
